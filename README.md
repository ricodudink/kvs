## KVS – Compacte Next.js site

Dit is een **compacte Next.js (App Router)** app, klaar voor deploy naar Vercel.

### Scripts

- `npm run dev` – start de dev server
- `npm run build` – build voor productie
- `npm start` – start de productie build

### Lokaal draaien

```bash
npm install
npm run dev
```

### Deploy naar Vercel

Koppel deze repository aan Vercel. Vercel detecteert automatisch Next.js.
